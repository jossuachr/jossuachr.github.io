DataFest Mizzou 2025 – Best in Show

As lead analyst for the RStudio Wizards team, I built a robust data pipeline and interactive dashboard on lease and market data, winning Best in Show.

Business Problem

Our industry partners needed clarity on rental pricing dynamics across multiple U.S. markets. The goals were to:

  • Discover factors influencing overall rent, including economic and occupancy indicators.
  • Predict lease values and categorize under-/over-valued contracts.
  • Enable stakeholders to explore scenarios via an interactive map and dashboard.

This required blending data engineering, statistical modeling, and geospatial visualization into a cohesive application.

Data Description

The analysis unified four datasets:

  • Leases: Detailed rent transactions and lease attributes.
  • Unemployment: Quarterly rates by state.
  • Price & Availability: Occupancy and availability metrics.
  • Major Market: Geographical definitions and demographic context.

Through grouping and summarization, we created overall_rent, imputed missing values, and engineered features like price_category.

Modeling & Tuning

We trained a Random Forest to predict rents, tuning mtry and min_n via cross-validation.

Show Modeling Code 
# Split data
set.seed(123)
data_split <- initial_split(grouped_data, strata=year)
train <- training(data_split); test <- testing(data_split)

# Recipe & spec
rf_recipe <- recipe(overall_rent ~ unemployment_rate + avg_occupancy_proportion + leasedSF + availability_proportion, data=train) %>%
  step_impute_median(all_numeric_predictors()) %>% step_normalize(all_numeric_predictors())

rf_spec <- rand_forest(mode='regression', trees=1000, mtry=tune(), min_n=tune()) %>% set_engine('ranger')

wf <- workflow() %>% add_recipe(rf_recipe) %>% add_model(rf_spec)

# Tuning
grid <- grid_random(mtry(range=c(2,10)), min_n(range=c(2,20)), size=30)
cv <- vfold_cv(train, v=5)
tune_res <- tune_grid(wf, resamples=cv, grid=grid, metrics=metric_set(rmse, rsq))

We finalized the workflow on full training data and evaluated on the test set to confirm generalization.

Key Insights

Our analysis provided:

  • A data-driven framework for accurate rent valuation (R²=0.85, RMSE=\$45).
  • Transparency into economic drivers via ALE plots.
  • An interactive map for spatial decision making.

We recommend deploying the model via a Plumber API and extending the dashboard for scenario testing of economic changes.
