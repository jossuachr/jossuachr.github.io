DataFest Mizzou 2025 – Best in Show

Leading the RStudio Wizards team, we delivered an end-to-end analysis pipeline on industry-provided data, culminating in interactive dashboards and actionable recommendations.

Data & Libraries

Our analysis was conducted entirely in R, leveraging:

  • tidyverse (dplyr, tidyr, ggplot2) for data wrangling and visualization
  • lubridate for date manipulation
  • iml for model-agnostic interpretation (ALE and importance)
  • ranger for efficient Random Forest modeling
  • sf and leaflet for geospatial mapping
  • shiny for the interactive dashboard

Data Cleaning & Manipulation

We merged three sources—leases, unemployment rates, and market definitions—by year, quarter, and state/market. Steps included:

  • Joining datasets with merge() to consolidate lease transactions with economic indicators.
  • Grouping to remove duplicate records and compute summary statistics per lease.
  • Imputing missing values using median or mode strategies for numeric and categorical features.
  • Feature engineering: calculated overall_rent and encoded price_category based on predicted vs. actual rent.

Random Forest Modeling

A Random Forest model (1,000 trees) was trained to predict rental prices and price category. We:

  • Split data into training and testing sets (80/20 stratified by price_category).
  • Tuned hyperparameters (mtry, min.node.size) via grid search.
  • Achieved R2 = 0.85 and RMSE = \$45 on holdout data.

Variable Importance

Extracted from the ranger model, the top predictors impacting rental price were:

  • median_income
  • unemployment_rate
  • lease_duration_months
  • market_size

Conclusions & Next Steps

Our analysis delivered robust rental price predictions, explained through ALE and VIP insights, and an engaging dashboard for stakeholders. To build on this work, we recommend:

  • Extending models with external economic forecasts and competitor data.
  • Deploying the Random Forest model via a Plumber API for real-time scoring.
  • Enhancing the Shiny dashboard with user-based scenario simulations.
  • Integrating customer feedback for continuous model retraining.
